# Sand Enderman Mod — Fabric 1.20.1

## Как собрать мод

### Требования
- **Java 17** (именно 17, не 21 и не 11)
- **Git** (опционально)

### Шаги сборки

1. Скачайте папку `sand-enderman-mod` полностью
2. Откройте терминал в этой папке
3. Запустите:
   ```
   gradlew build       # Windows
   ./gradlew build     # Linux / Mac
   ```
4. Готовый JAR будет в: `build/libs/sand-enderman-1.0.0.jar`
5. Скопируйте этот JAR в папку `.minecraft/mods/`

### Текстура босса (ВАЖНО)

Перед сборкой нужно создать текстуру жёлтых глаз:

1. Установите Pillow: `pip install Pillow`
2. Запустите: `python generate_texture.py`
3. Файл автоматически сохранится в нужную папку

**ИЛИ** вручную:
- Возьмите оригинальную текстуру эндермена из Minecraft (`enderman.png`)
- Замените белые пиксели глаз (координаты ~1,4 и 5,4) на жёлтый цвет #FFD200
- Сохраните как: `src/main/resources/assets/sandenderman/textures/entity/sand_enderman.png`

## Механики босса

### Нормальная стадия (100% — 20% HP)
| Параметр | Значение |
|----------|----------|
| Здоровье | 400 HP |
| Урон атаки | 9 HP |
| Смотришь на босса | Он замирает |
| Не смотришь | Телепортируется к тебе и атакует |
| Атакуешь замершего | +40% урона тебе обратно |

### Стадия ярости (< 20% HP = < 80 HP)
| Параметр | Значение |
|----------|----------|
| Урон атаки | 21 HP |
| Взгляд игрока | Игнорирует |
| Телепортация | Хаотично в радиусе 4 блоков раз в секунду |
| Заморозка при телепорте | 75% шанс на 1 секунду |
| Эффект при атаке | 60% шанс: слепота 5с или иссушение 5с (50/50) |
| Бросок вверх | 25% шанс при атаке, 50 блоков, кд 10 сек |

### В обеих стадиях
- Постоянные частицы жёлтого песка
- Полоса босса вверху экрана (жёлтая → красная в ярости)
- Спавн: только пустыня, естественный
